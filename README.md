## Components
- ESP8266 (Wemos D1 Min) x1
- ASK 433MHz PC-JC02 x1
- ASK 433MHz DX-RF x1
- Type-C USB TP4056 Battery Charger x1
- MicroSD card reader SPI x1
- Battery 4V x1 (optional)
